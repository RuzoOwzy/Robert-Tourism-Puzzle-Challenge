# Robert-Tourism-Puzzle-Challenge
An interactive “Play and Win” puzzle challenge web application . Users have a limited time to solve a Rubik’s Cube-style puzzle and, upon success, receive a unique Offer Code. The application features a live timer with millisecond precision, scramble and start/stop controls, pause functionality, and a responsive UI designed for desktop screens.

## Key Features:

- Interactive 3D puzzle using AnimCube3.js.

- Live countdown timer with milliseconds.

- Scramble, Start/Stop, and Pause controls.

- Responsive UI with clearly separated play and control panels.

- Offer Code generation and copy interface (placeholder ready for integration).

- User-friendly header with instructions and WhatsApp integration.

- Footer with copyright and developer credit.

## Technologies Used:

- HTML, CSS, JavaScript.

- AnimCube3.js for 3D cube functionality.

- Responsive layout with Flexbox.

- Google Fonts (Roboto) for typography.

## Usage:

- Open the application in a modern desktop browser.

- Read instructions and start the puzzle using the “Start” button.

- Solve the puzzle before the timer runs out to receive your Offer Code.

- Copy the Offer Code using the interface to claim rewards at Robert Tourism offices or via WhatsApp.

## Notes:

- The code generation and copy functionality is ready to integrate with backend logic.

- Designed to fit in a single desktop screen without vertical scrolling.
